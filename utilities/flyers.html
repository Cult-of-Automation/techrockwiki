Basics

Flying machines (flyers) are contraptions in Minecraft that can move whilst retaining it's original structure. Generally, these contraptions rely on slime (or honey) blocks to operate. Thus, the practice of engineering and working with flying machines is commonly know as 'slimestone'.

BE Disadvantages

In Bedrock Edition, several mechanics in Java Edition are lacking that render making/working with flying machines comparatively challenging:

Tile-entity update order

When two or more pistons are activated within the same redstone-tick (1/10 seconds), there is no way to determine the order in which said pistons will be activated if they are within the same chunk. This is due to the order of tile-entities (pistons), within a chunk, which is random. Update order between chunks is shuffled every 20gt.

Block-dropping

In Java Edition, sticky pistons, when activated by a pulse less than or equal to 1 redstone-tick, will act as a regular piston when pushing a block — in essense, not pulling the block back afterwards — but, if it hasn't push any blocks, will still pull blocks on retraction. This is because retraction happens instantaneously.

In Bedrock Edition, piston retraction is delayed by one redstone-tick. Thus, sticky pistons always pull blocks on retraction (if it is not blocked by unmovable blocks or push limit).

Quasi-connectivity & BUD

In java edition Quasi-connectivity (QC) is a property of pistons droppers and dispensers that allows them to be triggered by a update from certain blocks that occurs in a block adjacent to that component eg. (A piston above will cause the piston below to fire) this allows for more activation options for pistons in flying Machines. 

Block update detector (BUD) is a behaviour in java edition were pistons can be indirectly powered but won’t perform an action until they receive a block update. this mechanic is heavily used in flying machines as it allows for compactness and more powering options and can avoid the use of observers.

in bedrock edition neither of these mechanics are present so different approaches must be taken when building flying machine
